# my-website — Project Context

## What It Is
Marketing website for the Northstar Solana arbitrage bot.
Showcases the project, displays live paper-trading performance,
explains how DEX arbitrage works, and lets visitors join the community.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Vercel (hosting)

## Goals
1. Give Northstar a public face — clean, technical, trustworthy
2. Show live (or delayed) paper-trade P&L data
3. Link to GitHub, Discord, and Twitter
4. Grow an email waitlist for when live trading launches

## Pages
- `/` — hero, what it is, key stats, CTA
- `/how-it-works` — explain DEX arbitrage simply
- `/performance` — paper trade results (updated daily)
- `/blog` — technical posts from the marketer agent
