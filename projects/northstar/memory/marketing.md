# Marketing Context

## Brand Voice
- Technical and precise — we show the code, not just the promise
- Transparent — share real paper-trading P&L, wins and losses
- Community-first — help others learn DeFi arbitrage
- No hype — let results and open source speak for themselves

## Target Audiences
1. **Crypto traders** interested in DeFi / on-chain strategies
2. **Solana developers** looking for practical bot examples
3. **Quant/algo traders** exploring on-chain opportunities
4. **Open-source contributors** interested in Python + Solana

## Channels
| Channel | Purpose | Priority |
|---------|---------|----------|
| Twitter/X | Daily updates, paper trade results, threads | High |
| GitHub | README, docs, issues, discussions | High |
| Discord | Community Q&A, trading signals | Medium |
| Medium/Substack | Deep-dive technical articles | Medium |
| Reddit (r/solana, r/algotrading) | Sharing milestones and learnings | Low |

## Content Pillars
1. **Transparency** — publish paper trade logs, explain every decision
2. **Education** — how DEX arbitrage works, how to read the code
3. **Community** — highlight contributors, answer questions publicly
4. **Performance** — weekly P&L summaries, win rate, best/worst trades
